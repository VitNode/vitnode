## Update VitNode

VitNode works as NPM package, so you can update it using the following command:

import { Tab, Tabs } from 'fumadocs-ui/components/tabs';

<Tabs id='package-manager' persist items={['pnpm', 'npm']}>

```bash tab="pnpm"
pnpm add -D eslint-config-typescript-vitnode@latest --filter frontend && pnpm add -D eslint-config-typescript-vitnode@latest --filter backend &&  pnpm i vitnode-frontend@latest --filter frontend && pnpm i vitnode-backend@latest --filter backend
```

```bash tab="npm"
npm i -D eslint-config-typescript-vitnode@latest --workspace=frontend && npm i -D eslint-config-typescript-vitnode@latest --workspace=backend &&  npm i vitnode-frontend@latest --workspace=frontend && npm i vitnode-backend@latest --workspace=backend
```

</Tabs>

Packages has a script that will update the database, update the files, create a migration etc.

<Callout title="Canary">
  If you want to use the canary version, you can use `@canary` tag.
</Callout>

Sometimes in our [patch notes](https://github.com/VitNode/vitnode/releases) we will mention that you need to update a specific package for example `next` or `react` to a specific version.

VitNode is a Community Management Software built on top of [NextJS (React)](https://nextjs.org/), [NestJS (Express)](https://nestjs.com/) and [PostgreSQL](https://www.postgresql.org/).

**Community Management Software** is more than a simple CMS â€” it's a full-stack solution designed to empower developers with flexibility and ease. With built-in community features like user management, notifications, activity feeds, search content, device tracker, and more. **VitNode** is the perfect solution for building websites and applications that engage users and grow communities.

## Get Started

Create a new app with `create-vitnode-app`, it requires NodeJS 20+.

<Tabs id='package-manager' persist items={['pnpm', 'npm']}>

```bash tab="pnpm"
pnpm create vitnode-app@latest
```

```bash tab="npm"
npx create-vitnode-app@latest
```

</Tabs>

### Developemnt Server

To run the development server, use the following command:

<Tabs id='package-manager' persist items={['pnpm', 'npm']}>

```bash tab="pnpm"
pnpm dev
```

```bash tab="npm"
npm run dev
```

</Tabs>

## FAQ

import { Accordion, Accordions } from 'fumadocs-ui/components/accordion';

<Accordions>
  <Accordion title="Why NestJS insted of NextJS api?">
    The goal of `VitNode` is to provide a full-stack solution for developers to
    create websites with ease.

    `NextJS API` don't have a lot of features that `NestJS` have, like guards, interceptors, pipes, dependency injection, decorators, WebSockets, dynamic saving files etc.

    `NestJS` is a great framework for building APIs and `NextJS` is a great framework for building websites. Combining these two frameworks allows developers to create websites with ease.

  </Accordion>

    <Accordion title="When VitNode will be stable?">
    Not earlier then `NestJS 15` and `React 19` will be stable.
    </Accordion>

</Accordions>

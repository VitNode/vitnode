VitNode provides a codegen tool that generates TypeScript types for your GraphQL schema using the [GraphQL Code Generator](https://the-guild.dev/graphql/codegen) library.

## Create GraphQL File

VitNode detected all the GraphQL files _(with `.gql` extension)_ in your plugin inside `graphql` folder.

Example of a GraphQL file:

```graphql title="graphql/queries/test_example_query.gql"
query Test_example_query($cursor: Int, $first: Int) {
  test_example_query(cursor: $cursor, first: $first) {
    pageInfo {
      count
      endCursor
      hasNextPage
      hasPreviousPage
      startCursor
      totalCount
    }
    edges {
      id
      created
      name
    }
  }
}
```

## Generate Types

VitNode has build-in `codegen` script in `backend`. That mean when you make changes in `backend` code, the VitNode will automatically generate the types for you.

To generate types manualy for your GraphQL schema, run the following command:

import { Tab, Tabs } from 'fumadocs-ui/components/tabs';

<Tabs id='package-manager' persist items={['pnpm', 'npm']}>

```bash tab="pnpm"

```

```bash tab="npm"
npm run codegen
```

</Tabs>

## Internationalization (i18n)

As you can see, the title of plugin and navigation item is a language key. Add the language key inside the `langs` folder in `apps/frontend/plugins/{your_plugin_code}/langs/en.json` file in the frontend.

import { File, Folder, Files } from 'fumadocs-ui/components/files';
import { LanguagesIcon } from 'lucide-react';

<Files>
  <Folder name="frontend" defaultOpen>
    <Folder name="plugins" defaultOpen>
      <Folder name="{your_plugin_code}" defaultOpen>
        <Folder name="langs" defaultOpen>
          <File
            name="en.json"
            icon={<LanguagesIcon className="text-primary" />}
          />
        </Folder>
      </Folder>
    </Folder>
  </Folder>
</Files>

Default language is English `en`. If you have more then one language add keys to the other language files as well.

### Language Keys

- **Title Plugin** - key is always `admin_{your_plugin_code}.nav.title`.  
  For example: `blog.admin.nav.title`,
- **Navigation item** - key is always `admin_{your_plugin_code}.nav.{navigation_item_name}`.  
  For example: `blog.admin.nav.categories`.

Here is an example of the language keys:

```json title="en.json"
{
  "blog": {},
  "admin_blog": {
    "nav": {
      "title": "Blog",
      "categories": "Categories"
    }
  }
}
```

Now you can see your navigation item in the AdminCP.

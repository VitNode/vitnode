# Session

VitNode save device information like:

- IP address,
- Browser:
  - Name,
  - Version
- OS:
  - Name,
  - Version
- Device:
  - Type,
  - Vendor,
  - Model,
- Last seen

## Guest Session

We don't store any guest data.

## User Session

### Remember

User can choose to remember session. If user choose to remember session, session will be stored in database and will be valid for 90 days. After 90 days of inactivity session will be deleted and user will be logged out.

### No-remember

If user choose to not remember session, session will be stored max for 7 days. After 7 days session will be deleted and user will be logged out.

Session storage in cookie only for session duration. After session duration cookie will be deleted _(User closed the browser)_.

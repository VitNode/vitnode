# Color Input

This component is a simple color picker. You can use it to select a color.

## Example usage

```tsx {8, 18}
import {
  FormControl,
  FormField,
  FormItem,
  FormLabel,
  FormMessage,
} from 'vitnode-frontend/components/ui/form';
import { ColorInput } from 'vitnode-frontend/components/color/color-input';

return (
  <FormField
    control={form.control}
    name="color"
    render={({ field }) => (
      <FormItem>
        <FormLabel>Color</FormLabel>
        <FormControl>
          <ColorInput {...field} />
        </FormControl>
        <FormMessage />
      </FormItem>
    )}
  />
);
```

## Props

| Name                 | Type   | Default | Description                                         |
| -------------------- | ------ | ------- | --------------------------------------------------- |
| `disabled`           | `bool` | `false` | If `true`, the input will be disabled               |
| `disableRemoveColor` | `bool` | `false` | If `true`, the remove color button will be disabled |
